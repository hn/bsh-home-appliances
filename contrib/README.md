# B/S/H/ Home Appliances

## Home appliance models contributed by users

The files in this directory have been kindly contributed by dedicated users.
I cannot test these files or make any other statements about their function.
Credits go to the people referenced in the files.

Contributions to this collection are very welcome. Please feel free to submit your config file,
even if it only contains small changes or improvements â€” everyone can benefit from them.
The following is recommended:

- Create an issue where you describe your household appliance in as much detail as possible,
  including photos, the connectors used, any challenges encountered, and other relevant information
  ([Example 1](https://github.com/hn/bsh-home-appliances/issues/15), [Example 2](https://github.com/hn/bsh-home-appliances/issues/40)).

- Add your name to the header of the file and link to the issues or other helpful pages
  ([Example 1](bsh-dbus-wm14q441.yaml#L1-L11), [Example 2](bsh-dbus-wm14e3a0.yaml#L1-L10)).

- Create a [pull request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request) and submit the file.
